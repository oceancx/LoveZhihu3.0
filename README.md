# LoveZhihu3.0

去年在北京GDG活动中听了知乎设计师（Nova）的演讲后，一直觉得知乎3.0的风格深得MD的精髓（各种层级的关系体现的非常标准），于是就想着模仿出来。

个人能力有限，目前就做出了一个半成品，而且只兼容4.4以后的API，技术上没有难度，难的是怎么根据原App弄明白怎么做。
